# 味噌汁市ナレッジベース INDEX

このリポジトリは、Minecraft架空都市「味噌汁市」の公式設定・発信ノウハウを集約したナレッジベースです。
台本・原稿・記事を生成するAIは、作業前に必ずこのファイルを読んでください。

## AIへの利用ルール（必読）

1. このKBに書かれていない事実を創作しない。
2. `【要記入：…】` とある項目は「未確定情報」として扱い、推測で補完しない。出力に必要な場合は「（要確認）」と明示する。
3. `canon/` は事実、`voice/` は文体と型、`examples/` はお手本。役割を混同しない。
4. 固有名詞の読み方は `canon/glossary.md` に従う。glossaryにない固有名詞の読みを勝手に決めない。
5. 台本の出力形式は `voice/formats/` 配下の各フォーマット定義に従う。
6. 地区ファイルの `status` が「構想区域（凍結）」の地区について、新規開発が進んでいるかのような表現をしない。

## ディレクトリマップ

- `canon/` — 世界観の正典（事実のみを書く場所）
  - `city-overview.md` — 市の基本情報とコンセプト
  - `history.md` — 市史と、マイクラ現代建築界隈史における位置づけ
  - `glossary.md` — 用語と読み仮名（ナレーション・字幕の誤読防止を兼ねる）
  - `districts/` — 地区別ファイル（1地区1ファイル）
  - `landmarks/` — 主要建築物ファイル（1建物1ファイル）
- `voice/` — 発信の声
  - `style-guide.md` — 口調・テロップ規則・NG集
  - `formats/` — 動画フォーマット定義（台本の型と出力スキーマ）
- `examples/` — 成績メモ付きの過去台本（few-shotのお手本）
- `log/` — 月別の開発ログとネタの記録

## 更新ルール（人間向け）

- 新しい事実・ネタは log/ に記録し、編集会議で canon/ へ昇格させる。
- ファイルを更新したら、フロントマターの updated を更新日に変更する。
- 地区の status（開発区域／保存地区／構想区域（凍結））は都市計画マスタープランの決定と連動させる。
